# UserAuth

## Create DB
Run `python .\main.py` to create and populate example database.

## Setup
`python -m venv .venv` 
`.\\.venv\\Scripts\\Activate.ps1`
`pip install -r requirements.txt`

## Ready to run
`python app.py` To start webapp, terminal shows ip local ip address.
